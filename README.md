# 🏥 Queue Cure '26

## Problem
Many clinics still use paper tokens which causes long waiting times and no visibility.

## Solution
Queue Cure is a real-time digital clinic queue management system.

Receptionist can:
- Add patients
- Generate tokens
- Call next patient
- Handle emergency priority
- Track consultation history

Patients can:
- View current token
- See people ahead
- Check estimated waiting time


## Features

✅ Real-time queue updates  
✅ Socket.IO live synchronization  
✅ Emergency priority handling  
✅ Appointment booking  
✅ Queue export CSV  
✅ Dashboard analytics  


## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express.js
- Socket.IO


## How it works

Receptionist updates queue → 
Socket.IO sends update →
Patient screen updates instantly.
